
# What is LowPie?
> BEFORE WE DIVE IN, IT IS SAID "Low Pie" not "Loopy" OKAYY?? ty :3

LowPie is a new programming language that uses LUA syntax but in quite different way.

Here's a simple example:
```lua
Name = input("Enter your name: ")
print(Name)
print(fill("Your name is: ", Name, "!"))
-- The 'fill' function simply concatenates everything in order
-- output will be -> Your name is: NR!
```
